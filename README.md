# 🏥 Hospital Appointment Scheduler

A scalable backend application for managing hospital appointments, patients, and scheduling operations. Built using Spring Boot with secure authentication and PostgreSQL integration.

---

## 🚀 Live Demo

- 🌐 Frontend: https://symbdemo.netlify.app
- 🔙 Backend API: https://spring-hospital-scheduler-backend.onrender.com
---


## 🛠 Tech Stack

- **Backend:** Java, Spring Boot
- **Architecture:** RESTful APIs, Layered Architecture
- **Security:** JWT Authentication
- **Database:** PostgreSQL (Render)
- **Tools:** Git, Postman, Docker (optional)

---

## 📌 Features

- 📅 Book and manage appointments
- 👨‍⚕️ Doctor and patient management
- 🔐 Secure authentication using JWT
- ⚡ Scalable REST APIs
- 🗄 Persistent data storage with PostgreSQL

---

## 🧠 System Design

- Controller → Service → Repository architecture
- Clean separation of concerns
- Optimized database queries for better performance

---

## 🔗 API Endpoints (Sample)

| Method | Endpoint | Description |
|--------|--------|-------------|
| POST | `/api/auth/login` | User login |
| POST | `/api/appointments` | Create appointment |
| GET | `/api/appointments` | Get all appointments |
| DELETE | `/api/appointments/{id}` | Cancel appointment |
